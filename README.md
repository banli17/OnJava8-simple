# 《On Java 8》中文精简版

## 将书读薄

自己本身有 js 语言基础，需要 **快速** 学习 java，最终选择了这本书，但是本书 2000 多页，太长了，所以一边读一边删除大幅无用文字，保留自认为的有效部分。

- 第一次阅读(争取每个 .md 文档删减自不超过 500 行)

## 书籍简介

- 本书原作者为 [美] Bruce Eckel，即《Java 编程思想》的作者。
- 本书是事实上的 《Java 编程思想》第五版。
- 《Java 编程思想》第四版基于 JAVA **5** 版本；《On Java 8》 基于 JAVA **8** 版本。
- 精简自 [LingCoder 中文翻译版](https://github.com/LingCoder/OnJava8/blob/master/SUMMARY.md)。

感谢 🙏！
